# bun-postgres-drizzle

This project was created using `bun init` in bun v1.0.18. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
## To install dependencies:
```bash
bun install
```
```bash
bun run index.ts
```

## To build database
# generate schema
```bash
npm run generate
```
# run migration
```bash
cd db/schema
bun run index.ts 
```
